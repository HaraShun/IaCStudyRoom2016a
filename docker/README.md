# Docker 自習課題

``docker`` ディレクトリ以下を利用します。


## gitbucket

```
$ cd docker
$ docker-compose -f compose-gitbucket.yml build
$ docker-compose -f compose-gitbucket.yml up -d
```

## flask app on Docker

```
$ cd docker
$ docker-compose -f compose-flask.yml build
$ docker-compose -f compose-flask.yml up -d
```
